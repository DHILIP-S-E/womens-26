Security & Deployment

Production-grade security measures and cloud deployment architecture

Security Features

🔒

JWT Authentication

Secure token-based authentication with 7-day expiry. Protected routes require valid tokens.

🔐

Password Security

bcrypt hashing (10 salt rounds). Min 8 chars, uppercase + number required.

🛡️

Helmet Headers

X-Frame-Options, X-Content-Type-Options, HSTS enabled for XSS/clickjacking prevention.

⏱️

Rate Limiting

100 requests per 15 minutes per IP. Prevents brute force and DDoS attacks.

🔍

Input Validation

Zod schema validation on all inputs. Prisma ORM prevents SQL injection.

🌐

CORS Policy

Whitelist-based CORS. Only approved frontend origins can access the API.

👤

Anonymous Reporting

Incident reports can be filed anonymously. No user ID required.

🏥

FHIR Compliance

Health data export follows HL7 FHIR standards. Excludes private journals/chats.

☁️

AWS IAM

Least-privilege IAM roles for Lambda, SNS, Comprehend Medical, HealthLake access.
